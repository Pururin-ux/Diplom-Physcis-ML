# Слайд 1. Тема диплома и текущая логика проекта
- Тема: суррогатное моделирование низкоэнергетического спектра 2D квантовых точек (Kwant, tight-binding).
- Цель: получить методически корректный и воспроизводимый baseline в реалистичном объеме диплома.
- Структура работы: прямоугольный случай как валидационный benchmark, суперэллипсы как основной нетривиальный кейс.
- На текущем этапе акцент на корректности физики и протоколов оценки, а не на усложнении моделей.

# Слайд 2. Прямоугольный benchmark: верификация решателя
- Для прямоугольной точки выполнено сравнение Kwant с точным дискретным TB-решением.
- Итог: максимальная абсолютная ошибка по уровням E0..E3 составляет ~6.88e-14.
- Вывод: численный пайплайн и энергетическая конвенция работают корректно.
- Этот этап — контроль качества и валидация, а не заявка на научную новизну.

# Слайд 3. Аудит дискретизации суперэллипсов и выбор основной ветки
- Проведен sanity-аудит по параметру n: анализ гладкости E0..E3 и скачков числа узлов N_sites.
- Для непрерывного n выявлена чувствительность к лестничным артефактам дискретизации.
- Поэтому непрерывный n не выбран как основной регрессионный сюжет.
- Принято методическое решение: перейти к фиксированным дискретным классам формы n = {1.2, 2.0, 3.0, 4.0}.

# Слайд 4. Dense discrete-n датасет и baseline stress test
- Сформирован основной pre-ML датасет: 140 образцов (5 значений a × 7 aspect_ratio × 4 класса n).
- Протоколы стресс-теста: LOAO и LOARO; основные цели: E0, dE1, dE3; dE2 оставлен как диагностический.
- По milestone 06: E0 моделируется устойчиво, dE1 пригоден, dE3 сложнее, dE2 остается проблемным.
- LOAO стабильно более жесткий протокол, чем LOARO; крайние размеры a=24 и a=36 дают повышенные ошибки.

# Слайд 5. Ключевой результат 06b/06c, ограничения и ближайший шаг
- Ключевой результат: physics-informed признаки в Ridge дают лучшее качество, чем наивная параметризация [a, aspect_ratio].
- В 06c по winner matrix: для E0 стабильно выигрывает Set C, для dE1/dE3 чаще выигрывает Set B (иногда C в LOARO).
- dE2 не скрывается, но трактуется только как диагностическая цель для линейного baseline (не headline-критерий успеха).
- Ближайший шаг: аккуратно оформить главу baseline (метод, ограничения, таблицы) и затем переходить к следующему этапу без overclaiming.

## Короткий итог
- За отчетный период подтверждена корректность численной базы и получен консервативный, физически мотивированный Ridge-baseline для фиксированных классов суперэллипсов в валидированном диапазоне параметров.
